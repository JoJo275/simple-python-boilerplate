## Framework

- **pytest** with `pytest-cov` for coverage.
- Config in `pyproject.toml` under `[tool.pytest.ini_options]`.
- Run: `task test` or `hatch run test:run`.
- Matrix: Python 3.11–3.13.

## Conventions

- Unit tests in `tests/unit/`, integration tests in `tests/integration/`.
- Shared fixtures in `tests/conftest.py`.
- Be less strict about type annotations — don't require full annotations for mocks, fixtures, or test helpers.
- Don't require annotations for pytest fixtures unless they clarify intent.
- Test naming: `test_<function_or_behavior>` for functions, `Test<Class>` for classes.

## Pre-push

- pytest runs as a pre-push hook (`hatch run test`). All tests must pass before pushing.
- The test matrix (`hatch run test:run`) covers Python 3.11–3.13.
- `mkdocs>=1.6` is in the `test` dependency group because `test_include_templates.py` needs it to resolve `unittest.mock.patch` targets.
